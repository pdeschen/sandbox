Alert: Sample ui-state-error style.

Client-side templating showcase to support my associated blog post. While this is far from being complete, feel free fork. This is part of my web sandbox, experimenting with new concepts.

How it Works

  1. Both json model and mustache template view are fetched off the server.
  2. Mustache engine renders the view into HTML.
  3. HTML is dynamically replaced into div container below.

Hints

You may edit both view and model to your liking and hit the Apply button to get a new rendering.
